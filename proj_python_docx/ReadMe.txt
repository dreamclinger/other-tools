
（为了准备将SPSS输出的图片批量加入Word中）

This is a demo for creating / revising MS Word (*.docx) by Python

To run the demo, please:

1. place the folder at your Desktop

2. open a terminal (Application -> Utilities -> Terminal), enter: 

	cd ~/Desktop/proj_python_docx/

	sh setup.sh

3. to create demo.docx (in examples/demo.docx), in the terminal (step 2), enter:

	python create_demo.py

4. to search and edit demo.docx (in examples/*.jpg), please enter in the terminal: 

	python edit_demo.py
